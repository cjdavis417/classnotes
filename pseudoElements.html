CSS - Pseudo Elements

Note: As a rule, double colons (::) should be used instead of a single colon (:). This distinguishes pseudo-classes from pseudo-elements. However, since this distinction was not present in older versions of the W3C spec, most browsers support both syntaxes for the original pseudo-elements.

::first-line

::first-line


 p::first-line {
     font-weight: bold;
 }

::first-letter


 a nice drop cap effect
 p::first-letter {
     float: left;
     font-size: 80px;
     color: white;
     padding: 5px 10px;
     background: #384047;
     margin: 10px 10px 0 0;
     border-radius: 5px;
     line-height: 1px;
 }

::before and ::after

inserts before and after an element


 img::before {
     content: "hello"
 }

 img::after {
     content: "goodbye"
 }

attr()

This allows you to take an attribute from and element and display it on the web page.


 html
 <img title="jpg" src="img.jpg">

 css
 img::after {
     content: attr(title);
     display: inline-block;
 }
